== New Features and Enhancements

This version of Anypoint Platform Private Cloud Edition (Anypoint Platform for PCE) contains the following features and enhancements:

* Anypoint Platform for PCE features and enhancements

** TBD

* Features

** Anypoint Monitoring

*** Aggregate and map metrics.
*** Configure dashboards and alerts.
*** View application network graph with Anypoint Visualizer.
 
** Multiple IdP Support

*** Use native client provider (default) or configure an external client provider.
*** Assign IdPs by business groups, environment, and API visibility.

** API Groups

*** Support for API Groups across API Manager and Exchange.
*** Cohesively package API instances for API consumers.

* OAS 3.0 Support 
+
OAS3 is supported for API Designer, Anypoint Exchange, and Anypoint API Community Manager for creating, editing, and cataloging API specifications in OAS 3.0. Refer to xref:release-notes/platform/oas3.adoc[OpenAPI Specification 3.0 Support Release Notes] for additional information.

== Deprecations

The following API endpoints were deprecated on April 4th, 2020. Your action is required to avoid potential interruption of applications that integrate with these Anypoint Platform APIs.
 
This API change allows organization administrators to configure multiple Identity Providers (IdPs) for client management.

[%header%autowidth.spread]
|===
| API endpoints approaching deprecation | New API endpoints
|
accounts/api/organizations/:orgId/provider/clients
a|

* accounts/api/organizations/:orgId/clientProviders
* accounts/api/organizations/:orgId/clientProviders/:clientProviderId

|===

[%header%autowidth.spread]
|===
| API endpoints with changed behavior. For clients in third party Identity Providers only. | New API endpoints 
| GET, POST accounts/api/organizations/:orgId/clients
a|

* GET accounts/api/organizations/:orgId/clientProviders/:clientProviderId/clients
* GET, POST /exchange/api/v1/organizations/:orgId/applications

| GET, PATCH, DELETE accounts/api/organizations/:orgId/clients/:clientId
a|

*GET accounts/api/organizations/:orgId/clientProviders/:clientProviderId/clients/:clientId
* GET, PATCH, DELETE /exchange/api/v1/organizations/:orgId/applications/:clientId
|===

In addition, the following APIs responses no longer return `client_management` configuration properties.

[%header%autowidth.spread]
|===
| API endpoints | Deprecated properties
| GET accounts/api/organizations/:orgId | properties.identity_management
| PUT accounts/api/organizations/:orgId | properties.client_management
| GET accounts/api/organizations/:orgId | properties.client_management
| GET accounts/api/profile | organization.properties.identity_management
| GET accounts/api/me | organization.properties.identity_management
| GET accounts/api/organizations/:orgId/users | organization.properties.client_management
| GET accounts/api/organizations/:orgId/users/:userId | organization.properties.client_management
| GET accounts/api/users/me | organization.properties.identity_management
|===

To prevent disruptive changes, move to new endpoints.

Contact your CSM for assistance.
